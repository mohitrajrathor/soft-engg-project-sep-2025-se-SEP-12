# AURA - Academic Unified Response Assistant

Software Engineering Group Project Sept 2025

## Quick Start

### Backend
```bash
cd backend
.venv\Scripts\activate
python main.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
